Pete's Web App Sandbox from the JSP Book

In here we have a collection of dynamic content as per the JSP Book "Sevlets and Java Server pages (Wesley)"

JSP exmaples

  • Sandbox - this is a sam[ple jsp so that we can play around with it

Servlet examples

  • Dynamic Image Servlet - simple servlet that creates a really simple image
  • Request Headers - simple servlet to show the request headers passed from the web browser
  • Example Form - to show you how to debug a forms parameter list
  • Show File Content - to show the content of a file upload request
  • File Upload - to show the actual file upload process from a servlet
  • Linked Servlets - to show how two servlets can be linked with request scope objects
  • Servlet Context (Mock Error) - to show how to pass servlet context through to the servlet
  • Show Web App Source - to show the source behind this web app
